# Souled Store Capstone Project Documentation

This repository contains the capstone project documentation for the automation testing framework developed for The Souled Store website using Playwright with JavaScript.

## Project Objective
The objective of this project is to automate major functionalities of The Souled Store e-commerce application using Playwright and JavaScript. The framework is designed to improve testing efficiency, reduce manual effort, and ensure application quality across different browsers and devices.

## Modules Covered
- Homepage Testing
- Search Functionality
- Product Listing Page (PLP)
- Product Details Page (PDP)
- Cart Module
- Login Module
- Wishlist Module
- Checkout Flow
- Responsive Testing
- Cross-Browser Testing

Approximately 120 test cases will be automated as part of this project.

## Technologies Used
- JavaScript
- Playwright
- Node.js
- Visual Studio Code
- Git & GitHub
- HTML / Allure Reports
- GitHub Actions
